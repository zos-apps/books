## About

Digital library for reading and organizing your ebook collection.

## Features

- EPUB support
- Reading progress sync
- Bookmarks
- Highlights
- Library organization

## Installation

### From App Store
Open the **App Store** in zOS and search for "Books".

### Via npm
```bash
npm install @zos-apps/books
```

## Usage

- Add books to library
- Click to start reading
- Use bookmarks to save position



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
